# SSM Detailing - Professional Auto Detailing Services

Modern, responsive website for SSM Detailing, showcasing premium auto detailing services, portfolio, and customer reviews.

## Features
- Dynamic portfolio gallery with admin management
- Video reels showcase
- Customer testimonial system
- Comprehensive admin dashboard
- Supabase backend integration

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (Database + Storage + Auth)
